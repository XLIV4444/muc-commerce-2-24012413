## 用自己的话回答

- 什么是特征，什么是标签：特征就是用来做判断的已知信息，比如用户使用时长、有没有投诉；标签是我们想要预测的答案，这道题里就是用户是否流失。模型依靠特征去推算标签。
- 为什么要保留测试集：训练集用来给模型学习规律，测试集是模型从来没见过的数据。用测试集评估效果，可以检验模型能不能应对新样本，避免模型只会背训练数据、无法实际使用。
- 为什么83%准确率仍可能没有用：电商用户流失属于不均衡数据，大部分用户不会流失。模型只要全部预测 “不流失”，就能轻松拿到高准确率，但完全识别不出会流失的客户，无法完成业务目标，所以准确率高不代表模型有价值。
